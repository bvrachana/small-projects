CSS PSEUDO CLASSES

the css pseudo classes allow you to style the dynamic status of an element such as hover, active and focus state, as well as elements that are existing in the document tree but can't be targeted via the use of other selector without adding any IDs or Classes to them, for example targeting the first or last elements

A pseudo class start with colon(:), its syntax can be given with:

Selector:Pseudo-Class{Property:value}
